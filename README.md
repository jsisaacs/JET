# JET
JET is a command line tool for tracking your exercise.


+---------------------------------------------------------------------------------+
|                                   HELP DESK                                     |
+---------------------------------------------------------+-----------------------+
|                       [ Command ]                       |    [ Explanation ]    |
+---------------------------------------------------------+-----------------------+
|  -h                                                     |  help                 |
+---------------------------------------------------------+-----------------------+
|  -n objectName exercise sets reps weight date notes     |  new exercise object  |
+---------------------------------------------------------+-----------------------+
|  -d objectName                                          |  delete object        |
+---------------------------------------------------------+-----------------------+
|  -c                                                     |  clear all            |
+---------------------------------------------------------+-----------------------+
|  -s                                                     |  builds spreadsheet   |
+---------------------------------------------------------+-----------------------+
